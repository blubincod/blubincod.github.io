### resolver
- 필드를 실행 했을 때 실행되는 함수

⭐ user가 argument를 보낼 때 그 argument는 resolver function의 두번 째 argument가 된다.

```js
const tweets = [
  {
    id: "1",
    text: "hello",
  }
];

const typeDefs = gql`
  type Query {
    allTweets: [Tweet!]!
    tweet(id: ID!): Tweet
  }
`;

const resolvers = {
  Query: {
    allTweets() {
      return tweets;
    },
    tweet(root, args) {
      console.log(args);
      return null;
    },
  },
};

const server = new ApolloServer({ typeDefs, resolvers });
```
